# Data Analysis - Ultra Running Races

Analyse exploratoire de données (EDA) réalisée à partir d’un dataset réel, avec un focus sur les courses d’ultra-running aux États-Unis en 2020 (distances 50 km et 50 miles).

## Données
- Dataset : `TWO_CENTURIES_OF_UM_RACES.csv`

## Outils 
- Python
- pandas (data manipulation)
- seaborn & matplotlib (data visualization)

## Objectifs du projet
- Travailler sur un jeu de données réel et non nettoyé
- Mettre en œuvre une démarche complète de data analysis
- Extraire des informations pertinentes à l’aide de visualisations

## Compétences mises en œuvre
- Nettoyage de données (formats de dates, valeurs manquantes, doublons)
- Filtrage avancé (conditions multiples sur variables qualitatives et quantitatives)
- Feature engineering (calcul d’âge, vitesse moyenne)
- Transformation et harmonisation des variables (types, renommage, valeurs catégorielles)
- Analyse exploratoire (EDA)
- Visualisation comparative (distribution, histogrammes, violin plots)

## Analyses & visualisations
- Analyse de la répartition des distances (50km vs 50mi)
- Comparaison des participations selon le sexe
- Étude de la distribution de la vitesse moyenne
- Comparaison des performances selon la distance et le sexe
